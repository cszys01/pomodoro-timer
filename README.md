# 🍅 番茄钟

一款简洁的番茄专注计时器，帮助你高效管理时间。

## 功能

- **专注/休息交替** — 25分钟专注 + 5分钟短休息，每4轮后自动15分钟长休息
- **可视化进度环** — 圆弧进度条直观展示当前阶段进度
- **自动切换** — 可选择开启自动开始休息/专注
- **声音提醒** — 阶段结束时播放提示音
- **自定义设置** — 可调整各阶段时长、轮次间隔等参数
- **番茄计数** — 记录当天完成的番茄数量

## 环境要求

- Python 3.6+（需完整安装，非 Windows Store 快捷版）
- Windows 系统（使用 `winsound` 播放提示音）

## 快速启动

1. 确保已安装 Python（在终端运行 `python --version` 验证）
2. 双击 `pomodoro.py` 或在终端运行：

```bash
python pomodoro.py
```

## 配置

首次运行时自动生成 `config.json`，可修改以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `work_duration` | 专注时长（分钟） | 25 |
| `short_break` | 短休息时长（分钟） | 5 |
| `long_break` | 长休息时长（分钟） | 15 |
| `rounds_before_long` | 长休息间隔（个番茄） | 4 |
| `auto_start_breaks` | 自动开始休息 | true |
| `auto_start_work` | 自动开始专注 | false |
| `sound_enabled` | 启用提示音 | true |

也可以点击界面上的 **⚙ 设置** 按钮进行可视化修改。

## 操作

| 按钮 | 功能 |
|------|------|
| ▶ 开始 / ⏸ 暂停 | 开始或暂停计时 |
| ↺ 重置 | 重置当前阶段到初始时间 |
